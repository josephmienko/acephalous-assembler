HANDOFF CONSUMPTION (CROOKED-SENTRY-APPLIANCE)
─────────────────────────────────────────────────────────────────────────────

When acephalous-assembler completes:

1. Handoff JSON written to .coordination/
2. Contains: hostname, OS family/version, SSH user, network mode, status_server
3. Excludes: all secrets, password hashes, private keys

crooked-sentry-appliance should:
1. Read hostname, ssh_user, ip_address from handoff
2. Wait for first-boot via marker: /var/lib/acephalous-assembler/bootstrap-complete
3. SSH verify: ssh ${ssh_user}@${hostname}
4. Begin appliance provisioning (Frigate, backups, services)
5. Archive handoff for audit trail

BOUNDARY CONFIRMATION
─────────────────────────────────────────────────────────────────────────────
